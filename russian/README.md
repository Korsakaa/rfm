# RFM — Rust File Manager

Консольный двухпанельный файловый менеджер в стиле Total Commander, написанный на языке Rust.

## Описание

RFM работает в терминале на любом Linux-дистрибутиве.

**Особенности:**
- Две панели: левая и правая, независимая навигация
- Цветовая схема: папки — синий, файлы — зелёный, выделенные — жёлтый
- Настраиваемые горячие клавиши через файл `~/.config/fm/keymap.toml`
- Назначение программ для открытия файлов через `~/.config/fm/openers.toml`
- Меню USB-накопителей с автоопределением точек монтирования
- Поиск файлов по имени с навигацией вперёд/назад
- Изменение прав доступа (chmod) с поддержкой sudo и рекурсии
- Прогресс-бар при копировании и перемещении файлов

## Сборка

```bash
cargo build --release
```

## Установка

```bash
sudo cp target/release/rfm /usr/local/bin/rfm
```

## Запуск

```bash
rfm
```

## Конфигурация

Файлы конфигурации находятся в `~/.config/fm/`

| Файл | Описание |
|------|----------|
| `keymap.toml` | Горячие клавиши |
| `openers.toml` | Программы для открытия файлов по расширению (назначается из приложения — `Alt+Enter`) |

Пример `openers.toml`:
```toml
[openers]
pdf  = "evince"
mp4  = "mpv"
txt  = "kate"
png  = "eog"
zip  = "ark"
```

## Горячие клавиши

### Навигация

| Клавиша | Действие |
|---------|----------|
| `↑` / `↓` | Перемещение курсора |
| `Alt+↑` | Прыжок вверх (по умолчанию 5 строк) |
| `Alt+↓` | Прыжок вниз (по умолчанию 5 строк) |
| `PageUp` / `PageDown` | Прокрутка на страницу вверх/вниз |
| `g` | Перейти в начало списка |
| `G` | Перейти в конец списка |
| `Enter` / `→` | Войти в папку / открыть файл |
| `Backspace` / `←` | Выйти на уровень выше |
| `\` | Перейти в корневую директорию `/` |

### Панели

| Клавиша | Действие |
|---------|----------|
| `Tab` | Переключиться между панелями |
| `[` | Переключить фокус на левую панель |
| `]` | Переключить фокус на правую панель |

### Операции с файлами

| Клавиша | Действие |
|---------|----------|
| `Пробел` | Выделить / снять выделение с файла |
| `F5` | Копировать выделенные файлы в другую панель |
| `F6` | Переместить выделенные файлы в другую панель |
| `F7` | Создать новую папку |
| `F8` | Удалить выделенные файлы (с подтверждением) |
| `F2` | Создать новый файл |
| `r` | Обновить содержимое обеих панелей |

### Поиск

| Клавиша | Действие |
|---------|----------|
| `F3` | Открыть панель поиска |
| `Enter` | Найти следующее совпадение |
| `Esc` | Закрыть поиск |

### Права доступа

| Клавиша | Действие |
|---------|----------|
| `Alt+f` | Открыть диалог изменения прав (chmod) |
| `Пробел` | Переключить чекбокс |
| `Enter` | Применить (sudo при необходимости) |

### USB-накопители

| Клавиша | Действие |
|---------|----------|
| `Alt+u` | Открыть меню USB-накопителей |
| `Enter` | Перейти в точку монтирования |
| `Esc` | Закрыть меню |

### Прочее

| Клавиша | Действие |
|---------|----------|
| `Alt+Enter` | Назначить программу для открытия файла |
| `q` | Выйти из программы |

## Настройка клавиш

Пример `~/.config/fm/keymap.toml`:

```toml
[keys]
quit          = "q"
panel_switch  = "Tab"
panel_left    = "["
panel_right   = "]"
go_up         = "Backspace"
enter         = "Enter"
jump_up       = "Alt+Up"
jump_down     = "Alt+Down"
jump_amount   = 5
go_top        = "g"
go_bottom     = "G"
page_up       = "PageUp"
page_down     = "PageDown"
select        = "Space"
copy          = "F5"
move_files    = "F6"
mkdir         = "F7"
delete        = "F8"
create_file   = "F2"
search        = "F3"
chmod         = "Alt+f"
usb_menu      = "Alt+u"
refresh       = "r"
go_root       = "\\"
```

Поддерживаемые имена клавиш: `Enter`, `Tab`, `Backspace`, `Esc`, `Space`, `Delete`,
`Up`, `Down`, `Left`, `Right`, `Home`, `End`, `PageUp`, `PageDown`,
`F1`–`F10`, `Alt+X`, `Ctrl+X`, или любой одиночный символ.
